# The dataset correspond to underwater video data collecting in New Caledonia according the method described in
Pelletier, D., K. Leleu, D. Mallet, G. Hervé, G. Mou-Tham, M. Boureau, N. Guilpart. 2012. 
Remote high-Definition Rotating Video Enables Fast Spatial Survey of Marine Underwater Macrofauna and Habitats.
PLoS ONE 7(2): e30536. doi:10.1371/journal.pone.0030536
#
Dominique Pelletier, David Roos, Marc Bouchoucha, Thomas Schohn, William Roman, Charles Gonson, Thomas Bockel, 
Liliane Carpentier, Bastien Preuss, Abigail Powell, Jessica Garcia, Matthias Gaboriau, Florent Cadé, Coline Royaux, 
Yvan Le Bras, Yves Reecht. 2021. A standardized workflow based on the STAVIRO unbaited underwater video system for 
monitoring fish and habitat essential biodiversity variables in coastal areas. Frontiers in Marine Science 8, e689280. 
doi: 10.3389/fmars.2021.689280
#
# This dataset was used in the PAMPA project
Pelletier, D., E. Gamp, Y. Reecht & C. Bissery. 2011. 
Indicators of Marine Protected Area Effectiveness for conservation of coastal ecosystems and management of uses (PAMPA). 
Rapport scientifique final du projet PAMPA.  PAMPA/WP1/Coord/5. 58 p. Synthèse et Annexes. 
http://archimer.ifremer.fr/doc/00385/49601/
#
# The data format defined in this project for all the ecological dataset is used (see pampa-wp2-meth-3-formatdonneeswp2-240214.pdf)
#
# This format allows importation of data in the PAMPA interface wich computing biodiversity and marine resources indicators
Pelletier Dominique, Bissery Claire, Gonson Charles (2014). 
User guide for the PAMPA software. V2. IFREMER 12/2 212 911/F. 
https://archimer.ifremer.fr/doc/00256/36715/
